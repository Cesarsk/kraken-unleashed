# Privacy Policy

Effective date: April 1, 2026

Kraken Unleashed is an open source Windows desktop application for managing the LCD display on supported Kraken coolers. This privacy policy explains what data the application handles and when data may be sent to third-party services.

## Summary

- Kraken Unleashed does not require a user account.
- Kraken Unleashed does not include advertising.
- Kraken Unleashed does not include analytics or usage telemetry.
- Most app activity happens locally on the user's device.
- Optional in-app GIF search and download features may contact a third-party GIF service.

## Data Processed Locally

When you use Kraken Unleashed, the application may store the following data locally on your computer:

- GIF files that you choose to import or download into the app gallery
- saved placement presets for GIFs
- application state and local settings such as startup preferences

This local data is stored for the app to function and is not automatically transmitted to the project maintainer.

## Third-Party Network Requests

Kraken Unleashed can make network requests in the following cases:

### 1. In-app GIF search and download

If the in-app GIF search feature is available in your build, search queries and related requests are sent to the Klipy GIF service so the app can show search results and download selected GIF files.

Data that may be sent to Klipy includes:

- your search query
- technical request metadata normally sent by your system and network connection
- the URL request needed to download a GIF you selected

Klipy is a third-party service and its handling of data is governed by its own terms and privacy practices.

### 2. External links opened by the user

If you click links in the app, Kraken Unleashed may open external websites in your default browser. Any data handled after that point is governed by the destination site's own policies.

## Device Communication

Kraken Unleashed communicates locally with supported Kraken hardware over USB in order to detect devices, adjust settings, recover the display state, and transfer GIF content. This device communication does not by itself send your data to the project maintainer.

## Release Scanning

Project release artifacts may be uploaded to VirusTotal by the maintainer during the release process for malware scanning. This applies to packaged release files created by the maintainer and does not result from end-user runtime activity inside the application.

## Data Sharing

Except for the third-party requests described above, Kraken Unleashed does not automatically send personal data or app usage data to the maintainer or to other third parties.

## Your Choices

- You can avoid third-party GIF search requests by not using the in-app search feature.
- You can use locally stored GIF files instead of downloading GIFs through the app.
- You can review the source code of the project at the public repository to verify app behavior.

## Data Retention

Local files and settings remain on your device until you remove them or uninstall the application. Third-party services may retain request data according to their own policies.

## Changes To This Policy

This privacy policy may be updated when project behavior changes. The latest version should be published in the project repository.
