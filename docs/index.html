Kraken Unleashed is a focused Windows desktop app for getting great-looking GIFs onto supported Kraken LCD coolers. Detect the screen, position the asset, deploy it, and move on.

Contributing

Real hardware validation is the best contribution.

If you are validating a new model, the most useful details are the exact cooler name, USB VID and PID, detected resolution, and whether detection, brightness, recovery, and GIF deploy all work.

Build from source

Local Windows workflow.

Repo root
npm install
npm run backend:stage
npm start

Requirements: Windows, Node.js with npm, the Rust toolchain with cargo, and a supported Kraken LCD over USB. For release packaging, use Node.js 22.x to match CI.

Safety notes

Direct writes to the LCD device.

  • This app writes directly to the display hardware.
  • Use it at your own risk.
  • Keep competing control software closed while deploying.
Trust and signing

Signing is still pending.

Free code signing has been requested through SignPath.io with a certificate from SignPath Foundation. Until that is approved, released binaries remain unsigned.
