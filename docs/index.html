Supported Devices

Validated first, broader support next.

  • Kraken Elite RGB 2024 / Kraken Elite V2 Validated in the app with PID `0x3012`.
  • Kraken Elite 2023 Listed in compatibility view with supported backend path.
  • Kraken Z3 Legacy support path remains listed for compatibility checks.

Community validation is welcome for additional models and device revisions.

Workflow

What using the app looks like.

  1. Launch the app and let it detect the connected LCD.
  2. Upload a GIF and select it from the local gallery.
  3. Use the editor to adjust zoom, pan, and rotation.
  4. Deploy the prepared GIF directly to the cooler display.
  5. Use Restore Liquid Screen if the LCD needs a clean reset.

Native Backend

Current device-facing commands.

  • infoDetect a supported Kraken LCD and report resolution.
  • brightnessSet LCD brightness from 0 to 100.
  • recoverReturn the screen to liquid display mode.
  • writePrepare and transfer a GIF directly to the device.

Trust And Safety

What the project does and does not do.

  • This app writes directly to the LCD device over USB.
  • There is no analytics or usage telemetry built into the app.
  • Most activity stays local unless you use optional GIF search.
  • Keep competing control software closed while deploying.
