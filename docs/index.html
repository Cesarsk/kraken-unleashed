Third-party GIF utility

Custom GIFs on supported Kraken LCD coolers.

Kraken Unleashed is a small desktop utility for getting great-looking GIFs onto supported Kraken LCD coolers. Detect the screen, frame the asset, deploy it, and recover the display from the same app.

Supported Devices

Validated models first.

  • Kraken Elite RGB 2024 / Kraken Elite V2 Validated in the app with PID `0x3012`.
  • Kraken Elite 2023 Listed in compatibility view with PID `0x300C`.
  • Kraken Z3 Listed in compatibility view with PID `0x3008`.

More device support is planned, not necessarily limited to Kraken coolers.

Contributing

Hardware validation is especially useful.

Contributions are welcome, especially for expanding cooler support. Exact model name, USB VID/PID, resolution, and notes about detection, brightness, recovery, and GIF deploy behavior are the most useful contributions.

Build From Source

Run the app locally on Windows.

Local run 
npm install
npm run backend:stage
npm start

You need Windows, Node.js with npm, the Rust toolchain with cargo, and a supported Kraken LCD connected over USB. For release packaging, use Node.js 22.x to match CI.

Safety Notes

Direct device writes.

  • This app writes directly to the LCD device.
  • Use it at your own risk.
  • Keep competing control software closed while deploying.
