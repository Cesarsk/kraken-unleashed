Privacy Policy

Kraken Unleashed privacy summary.

Most app activity happens locally on the user's device. The only routine third-party network requests come from optional features like in-app GIF search and download.

Effective date: April 1, 2026

Summary

  • Kraken Unleashed does not require a user account.
  • Kraken Unleashed does not include advertising.
  • Kraken Unleashed does not include analytics or usage telemetry.
  • Most app activity happens locally on the device.
  • Optional GIF search and download features may contact a third-party GIF service.

Data Processed Locally

  • GIF files that you import or download into the app gallery
  • Saved placement presets for GIFs
  • Application state and local settings such as startup preferences

This data is stored locally so the app can function and is not automatically transmitted to the maintainer.

Third-Party Network Requests

1. In-app GIF search and download

If the in-app GIF search feature is available in your build, search queries and related requests are sent to the Klipy GIF service so the app can show search results and download selected GIF files.

  • Your search query
  • Technical request metadata normally sent by your system and network connection
  • The request needed to download a GIF you selected

2. External links opened by the user

If you click links in the app, external websites may open in your default browser. Data handled after that point is governed by the destination site's own policies.

Device Communication

Kraken Unleashed communicates locally with supported Kraken hardware over USB in order to detect devices, adjust settings, recover display state, and transfer GIF content. That device communication does not by itself send your data to the maintainer.

Release Scanning

Project release artifacts may be uploaded to VirusTotal by the maintainer during the release process for malware scanning. This applies to packaged release files created by the maintainer and does not result from runtime activity inside the application.

Your Choices

  • Avoid third-party GIF search requests by not using the in-app search feature.
  • Use locally stored GIF files instead of downloading through the app.
  • Review the source code in the public repository to verify app behavior.
