Privacy Policy

Kraken Unleashed privacy summary.

Most app activity happens locally on the user's device. Optional in-app GIF search and download features may contact a third-party GIF service.

Effective date: April 1, 2026

Summary

  • Kraken Unleashed does not require a user account.
  • Kraken Unleashed does not include advertising.
  • Kraken Unleashed does not include analytics or usage telemetry.
  • Most app activity happens locally on the user's device.
  • Optional in-app GIF search and download features may contact a third-party GIF service.

Data Processed Locally

  • GIF files that you choose to import or download into the app gallery
  • Saved placement presets for GIFs
  • Application state and local settings such as startup preferences

Third-Party Network Requests

If the in-app GIF search feature is available in your build, search queries and related requests are sent to the Klipy GIF service so the app can show search results and download selected GIF files.

  • Your search query
  • Technical request metadata normally sent by your system and network connection
  • The URL request needed to download a GIF you selected

Device Communication

Kraken Unleashed communicates locally with supported Kraken hardware over USB in order to detect devices, adjust settings, recover the display state, and transfer GIF content.

Release Scanning

Project release artifacts may be uploaded to VirusTotal by the maintainer during the release process for malware scanning. This applies to packaged release files created by the maintainer and does not result from end-user runtime activity inside the app.

Data Sharing

Except for the third-party requests described above, Kraken Unleashed does not automatically send personal data or app usage data to the maintainer or to other third parties.

Your Choices

  • You can avoid third-party GIF search requests by not using the in-app search feature.
  • You can use locally stored GIF files instead of downloading GIFs through the app.
  • You can review the source code of the project at the public repository to verify app behavior.
