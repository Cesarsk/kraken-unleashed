Kraken Unleashed
Product Hunt GitHub

Windows desktop utility

Put better GIFs on your Kraken LCD.

Deploy custom GIFs directly over USB, tune the fit for the circular screen, and keep a local gallery ready for the next swap.

Download for Windows See it run View on Product Hunt

Built for the LCD

Fast deploys without vendor control software.

Kraken Unleashed focuses on one job: getting a GIF onto a supported Kraken display with fewer steps and enough control to make it look right.

Fit controls

Supported hardware

Validated for Kraken Elite RGB 2024 and Kraken Elite V2.

Kraken Elite 2023 and Kraken Z3 are listed in the compatibility view. Community validation is welcome for broader cooler support.
